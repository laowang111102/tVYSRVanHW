# 花卉识别分类系统 python426

## 项目概述

花卉识别分类系统基于 Python、ResNet50 和 PyTorch，实现了对七种花卉的识别及其历史记录功能。

## 技术栈

- **开发环境**：PyCharm、Anaconda
- **编程语言**：Python 3.9
- **依赖库**：OpenCV、PyQt5、PyTorch

## 功能模块

- **花卉识别**：使用 ResNet50 网络进行图像分类，当前支持七种花卉识别。
- **历史记录**：保存用户的识别历史，便于回顾和分析。

## 系统角色

- **用户**：通过 GUI 界面进行花卉图片的上传与识别。
- **系统**：接收用户上传的图片，使用预训练模型进行花卉分类。

## 运行环境

- Python 3.9
- OpenCV
- PyQt5
- PyTorch

## 部署步骤

1. 确保所有依赖库正确安装。
2. 使用提供的训练好的权重启动系统。
3. 运行 GUI 源码，启动花卉识别分类系统。

## 目录结构

```
花卉识别分类系统/
├── data/             # 数据集目录

├── src/              # 源码目录
│   ├── gui.py        # GUI 源码
│   └──识别.py        # 识别逻辑源码
└── logs/             # 训练日志
```

## 核心亮点

- **预训练模型**：已包含训练好的网络模型，用户可以直接运行识别系统。
- **可扩展性**：基于 PyTorch 的 ResNet50 网络可适用于其他分类问题，用户也可以自行重新训练。
- **数据集丰富**：包含 11,200 张图片的数据集，帮助用户进行有效的模型训练。
- **实验报告**：提供详尽的实验报告，帮助用户深入理解系统的实现原理。
